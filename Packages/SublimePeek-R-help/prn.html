prnzR Documentation

Print and Object with its Name

Description

Prints an object with its name and with an optional descriptive text string. This is useful for annotating analysis output files and for debugging.

Usage

 prn(x, txt)

Arguments

x

any object

txt

optional text string

Side Effects

prints

See Also

print, cat

Examples

 x <- 1:5
 prn(x)
