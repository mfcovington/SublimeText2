loadMeuseR Documentation

load the Meuse data set

Description

load the Meuse data set

Usage

 loadMeuse(gridded = TRUE)

Arguments

gridded

logical; if FALSE, meuse.grid will be imported as SpatialPointsDataFrame, if TRUE as SpatialPixelsDataFrame

Value

none; it has a side effect that meuse and meuse.grid will be saved to the global environment

See Also

meuse, meuse.grid

Examples

 library(sp)
 loadMeuse()
 summary(meuse)
 summary(meuse.grid)
