time_transR Documentation

Transformation for times (class POSIXt).

Description

Transformation for times (class POSIXt).

Usage

   time_trans()

Examples

 hours <- seq(ISOdate(2000,3,20, tz = ""), by = "hour", length.out = 10)
 t <- time_trans()
 t$trans(hours)
 t$inv(t$trans(hours))
 t$format(t$breaks(range(hours)))
