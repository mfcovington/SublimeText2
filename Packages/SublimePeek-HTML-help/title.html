title

Summary

The HTML Title Element (<title>) defines the title of the document, using shown in a browser's title bar or on the page's tab. It can only contain text and any contained tags is not interpreted.

Usage context

Content categories Metadata content.
Permitted content Text
Tag omission All tags required. Note that leaving off </title> should cause the browser to ignore the rest of the page.
Permitted parent elements A <head> element that contains no other <title> element.
Normative document HTML 5, section 4.2.2, HTML 4.01, section 7.4.2

Attributes

Like all other HTML elements, this element supports the global attributes. It does not support any other attribute.

DOM interface

This element implements the HTMLTitleElement interface.

Example

<title>Awesome page title</title>
