details

Summary

The HTML details element (<details>) is used as a disclosure widget from which the user the retrieve additional information.

Usage context

Permitted content One <summary> element followed by flow content
Tag omission None, both the start tag and the end tag are mandatory.
Permitted parent element Any element that accepts flow content.
Normative document HTML5, section 4.11.1

Attributes

Like all other HTML elements, this element has the global attributes.

open
This Boolean attribute indicates whether the details will be shown to the user on page load. If omitted the details will be hidden.

DOM interface

This element implements the HTMLDetailsElement interface.

Examples

<details>  <summary>Some details</summary>  <p>More info about the details.</p></details>
