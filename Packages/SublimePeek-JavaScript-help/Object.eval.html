Object.eval

Summary

Evaluates a string of JavaScript code in the context of an object.

Syntax

object.eval(string)

Parameters

string 
Any string representing a JavaScript expression, statement, or sequence of statements. The expression can include variables and properties of existing objects.

Description

The eval method can't no longer be used as a method of Object. Use the top-level eval function instead.
